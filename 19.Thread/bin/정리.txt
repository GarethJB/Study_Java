main 메소드로부터 시작된 명령문들이 순차적인 흐름을 타고 처리가 된다.
이러한 실행흐름을 쓰레드 라고 한다
프로세스 한 개가 생성되고 main에서 시작해 main으로 끝나는 실행흐름이 한개인
싱글쓰레드 프로그램.
멀티쓰레드 프로그램: 하나의 프로세스가 여러 실행흐름을 갖는 프로그램

main 은 자동실행되어지지만 다른 쓰레드는 자동실행되지 않는다.
main 에서 쓰레드를 실행시켜줘야 한다.
main 쓰레드가 종료되어도 각각 쓰레드들은 실행흐름이 모두 끝나야
최종적으로 프로그램 종료가 된다.

멀티쓰레드 프로그램 만들어보기
1. 동시진행 작업 준비
2. 각 작업을 쓰레드로 만든다. → Thread 를 상속받기, Runnable 를 구현하기
3. 쓰레드 내부에 구체적인 처리를 구현한다. → run() 메소드에 구현
4. 쓰레드가 실행되도록 main에서 시작시킨다. → start() 메소드로 호출

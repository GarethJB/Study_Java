데이터를 읽어들이는 처리 : in
데이터를 내보내는 처리 : out

PrintStream out;
자바에서의 데이터 입출력 : 스트림 형태로 이루어진다(입력/출력스트림)

데이터 입출력 스트림 단위 : 바이트/문자 인지에 따라 기본스트림이 있다
입력스트림 : InputStream(바이트) / Reader(문자)
출력스트림 : OutputStream(바이트) / Writer(문자)

데이터를 파일로 내보내고, 읽어들이고
파일로의 입/출력이 FileInputStream(바이트) / FileReader(문자)
					FileOutputStream(바이트) / FileWriter(문자)
					
메모장에 메모(쓰기) : 파일을 연다 → 내용을 쓴다 → 파일을 닫는다
메모장 보기(읽기) : 파일을 연다 → 내용을 읽는다 → 파일을 닫는다
